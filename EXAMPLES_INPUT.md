# 📝 Примеры ввода текста в боте

## 🎖️ Регистрация ФИО

### 📋 Требования к формату:
- Фамилия с заглавной буквы
- Инициалы имени и отчества через точку
- Минимум 5 символов
- Хотя бы одна точка в строке

### ✅ Правильные примеры:

#### 👤 Базовые примеры:
- `Иванов И.И.` - стандартный формат
- `Петров П.П.` - с одинаковыми инициалами
- `Сидоров С.С.` - другой пример
- `Козлов К.К.` - короткая фамилия

#### 👥 Сложные фамилии:
- `Петров-Иванов П.И.` - двойная фамилия
- `Васильев В.В.` - длинная фамилия
- `Смирнов С.А.` - разные инициалы
- `Соколов С.М.` - стандартный формат

#### 🌍 Национальные особенности:
- `Айдаров А.А.` - восточная фамилия
- `Белоусов Б.Б.` - составная фамилия
- `Дубинин Д.Д.` - окончание на -ин
- `Жуков Ж.Ж.` - редкая первая буква

### ❌ Неправильные примеры:

#### 🚫 Неверный формат:
- `Иванов Иван` - полное имя вместо инициалов
- `И.И. Иванов` - неправильный порядок
- `иванов и.и.` - строчные буквы
- `Иванов И И` - отсутствуют точки

#### 🚫 Недостаточно символов:
- `И.И.` - только инициалы
- `Ли Л.` - слишком короткая фамилия
- `А А.А.` - пробел в фамилии
- `123 И.И.` - цифры в фамилии

#### 🚫 Неправильные символы:
- `Иванов И.И` - отсутствует последняя точка
- `Иванов И.И.!` - лишний символ
- `Иванов И О` - отсутствуют точки
- `Иванов И.И.` - лишний пробел

## 🛠️ Валидация системы

### 🔍 Проверки при вводе:
1. **Минимальная длина**: не менее 5 символов
2. **Наличие точки**: хотя бы одна точка в строке
3. **Формат**: Фамилия + пробел + инициалы
4. **Уникальность**: ФИО не должно повторяться

### 🔄 Автоматическая обработка:
- Удаление лишних пробелов
- Проверка на дубликаты
- Валидация формата
- Сохранение в базе данных

## 🎯 Советы по вводу

### ✅ Рекомендации:
1. **Используйте стандартный формат**: Фамилия И.О.
2. **Проверяйте точки**: обязательны после инициалов
3. **Заглавные буквы**: фамилия с большой буквы
4. **Один пробел**: между фамилией и инициалами

### 🚀 Быстрый ввод:
- Скопируйте пример и измените на свой
- Проверьте наличие всех точек
- Убедитесь в правильности заглавных букв
- Нажмите отправить

## 📱 Интерфейс бота

### 🎯 Сценарий успешной регистрации:
1. Пользователь отправляет `/start`
2. Бот просит ввести ФИО с примерами
3. Пользователь вводит корректное ФИО
4. Бот подтверждает регистрацию
5. Показывается главное меню

### ⚠️ Сценарий с ошибкой:
1. Пользователь вводит неправильный формат
2. Бот показывает детальную ошибку
3. Предлагает правильные примеры
4. Пользователь исправляет ввод
5. Регистрация завершается успешно

## 🔧 Для разработчиков

### 📋 Функции валидации:
```python
def validate_name(text):
    # Минимум 5 символов
    if len(text) < 5:
        return False
    
    # Хотя бы одна точка
    if '.' not in text:
        return False
    
    return True
```
