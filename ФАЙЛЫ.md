# 📁 Что за файлы в проекте?

## 🚀 Главные файлы для запуска:
- **`1_setup.py`** - Настройка и установка (запускать ОДИН раз)
- **`2_start.sh`** - Запуск бота (можно запускать вручную)
- **`3_auto_restart.py`** - Автоматический перезапуск каждые 55 минут (для 24/7)

## ⚙️ Настройки:
- **`.env`** - Настройки бота (токен, ID админа)
- **`requirements.txt`** - Список программ для установки

## 🤖 Основные файлы бота:
- **`main.py`** - Главная программа бота
- **`handlers.py`** - Обработка команд
- **`database.py`** - Работа с базой данных
- **`utils.py`** - Вспомогательные функции
- **`admin.py`** - Админ панель
- **`keyboards.py`** - Кнопки в боте
- **`notifications.py`** - Уведомления

## 🔧 Служебные файлы:
- **`backup.py`** - Резервные копии
- **`monitoring.py`** - Мониторинг
- **`export.py`** - Экспорт данных
- **`check_database.py`** - Проверка базы данных
- **`create_database.py`** - Создание базы данных
- **`config.py`** - Конфигурация
- **`pythonanywhere_support.py`** - Поддержка PythonAnywhere

## 📋 Документация:
- **`КАК_ЗАПУСТИТЬ_БОТА.md`** - Главная инструкция
- **`README.md`** - Техническая документация
- **`ФАЙЛЫ.md`** - Этот файл

## 🎯 Что запускать?
1. **Первый раз:** `python3 1_setup.py`
2. **Для 24/7:** `python3 3_auto_restart.py`
3. **Вручную:** `./2_start.sh`
