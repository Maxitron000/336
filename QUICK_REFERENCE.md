# 🚀 Быстрый справочник - Исправление токена бота

## 💡 Основная проблема
**Токен бота недействителен** → Нужен новый токен от @BotFather

## 🔑 Быстрое решение (5 минут)

### 1. Получить новый токен
```
Telegram → @BotFather → /mybots → [выбрать бота] → API Token
```

### 2. Проверить токен
```bash
source venv_bot/bin/activate
python validate_token.py [НОВЫЙ_ТОКЕН]
```

### 3. Обновить конфигурацию
```bash
nano .env
# Заменить BOT_TOKEN=... на новый токен
```

### 4. Проверить подключение
```bash
python test_bot_connection.py
```

### 5. Запустить бота
```bash
python main.py
```

## 🛠️ Полезные команды

| Команда | Описание |
|---------|----------|
| `python validate_token.py` | Проверка токена (интерактивно) |
| `python validate_token.py TOKEN` | Проверка конкретного токена |
| `python test_bot_connection.py` | Полная диагностика подключения |
| `nano .env` | Редактирование конфигурации |
| `source venv_bot/bin/activate` | Активация виртуального окружения |
| `python main.py` | Запуск бота |

## ⚠️ Если что-то не работает

### Токен не принимается:
- Убедитесь, что скопировали полностью
- Проверьте, что нет лишних пробелов
- Убедитесь, что бот не был удален

### Бот не запускается:
- Проверьте логи: `tail -f logs/bot.log`
- Убедитесь, что виртуальное окружение активно
- Проверьте ADMIN_IDS в .env

### Создать нового бота:
```
@BotFather → /newbot → [название] → [username_bot] → [получить токен]
```

## 📋 Формат правильного токена
```
1234567890:ABCDEF1234567890abcdef1234567890ABC
```
- Содержит `:` 
- Длина ~45-50 символов
- Начинается с цифр
