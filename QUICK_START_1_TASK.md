# 🚀 БЫСТРЫЙ ЗАПУСК (1 Scheduled Task в сутки)

## ⚡ За 3 минуты до готовой работы 24/7!

### 1️⃣ **Простая настройка:**
```bash
python3 setup_daily.py
```

### 2️⃣ **Введите данные:**
- 🔑 **BOT_TOKEN** (получите у @BotFather)
- 🆔 **Ваш Telegram ID** (узнайте у @userinfobot)

### 3️⃣ **Настройте Scheduled Task в PythonAnywhere:**
- **Dashboard** → **Tasks** → **Scheduled**
- **Command:** `/home/ваш_username/telegram_bot/start_daily.sh`
- **Schedule:** `1 раз в сутки` (любое время)
- **Description:** `Telegram Bot Daily Runner`

## ✅ **Готово!** Бот работает 24/7!
