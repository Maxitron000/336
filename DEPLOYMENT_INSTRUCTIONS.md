```bash

# Название: military-telegram-bot
# Описание: 🛡️ Telegram Bot for military unit personnel management

```

### 2. Подключите к удаленному репозиторию

```bash
# Замените YOUR_USERNAME на ваше имя пользователя GitHub
git remote add origin https://github.com/YOUR_USERNAME/military-telegram-bot.git

# Переименуйте ветку в main
git branch -M main

# Выгрузите код
git push -u origin main
```

### 3. Проверьте результат

1. Перейдите в созданный репозиторий
2. Убедитесь, что все файлы загружены
3. Проверьте README файлы

## 🔧 Настройка после выгрузки

### 1. Создайте файл .env

```env
BOT_TOKEN=your_telegram_bot_token_here
ADMIN_IDS=123456,789012
DB_PATH=data/bot_database.db
LOG_LEVEL=INFO
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Запустите миграцию

```bash
python3 migrate_roles.py
```

### 4. Запустите бота

```bash
python3 main.py
```
