# 🎖️ Telegram Bot для учета персонала v2.0

**Военная система учета персонала с максимальной защитой от дурака и интуитивным интерфейсом**

## 🚀 Новые возможности v2.0

### 🎯 **Военная тематика и статусы:**
- 🟢 **В расположении** - боец прибыл в часть или не было отметки убытия
- 🔴 **Вне расположения** - боец убыл из части
- 📝 **Локация "Другое"** - ручной ввод места убытия
- 🎖️ **Военные эмодзи** - красные/зеленые кружки для прибытия/убытия

### 👑 **Система ролей:**
- 👤 **Боец** - обычный военнослужащий
- 👑 **Админ** - администратор с расширенными правами
- 👑 **Главный админ** - полные права управления

### 📊 **Улучшенные сводки:**
- 📈 **Сегодняшняя сводка** - кто в расположении/вне расположения
- 🕐 **Автоматическая сводка в 19:00** - ежедневный отчет
- 📊 **Excel с цветовой индикацией** - зеленые/красные строки

### 🔔 **Система уведомлений:**
- 📢 **Массовые уведомления** - отправка всем вне расположения
- 🕐 **Автоматические напоминания** - 19:00 и 20:30
- 💬 **Креативные тексты** - 50+ братских фраз

### 🛡️ **Защита от дурака:**
- ⚠️ **Двойное подтверждение** - для опасных операций
- 🔐 **Проверка прав** - на каждом уровне
- 📝 **Логирование** - всех действий
- 🚨 **Предупреждения** - с описанием последствий

## 🎯 **Основные функции**

### 👤 **Для бойцов:**
- 📍 Отметка местоположения (прибыл/убыл)
- 📝 Ручной ввод места убытия
- 📊 Просмотр собственного статуса
- 📖 Журнал своих действий

### 👑 **Для админов:**
- 📊 Быстрая сводка с эмодзи
- 👥 Управление личным составом
- 📖 Журнал событий с фильтрами
- 📢 Массовые уведомления
- 📥 Экспорт в Excel с цветами

### 👑 **Для главных админов:**
- 🚨 Отметить всех прибывшими
- 👑 Управление ролями
- ⚙️ Системные настройки
- 🛠️ Обслуживание системы

## 📱 **Использование**

### 👤 **Для бойца:**
1. 🚀 `/start` - запуск бота
2. 📍 "Отметить местоположение" - выбор локации
3. ✅ "Прибыл" / 🔴 "Убыл" - отметка действия
4. 📝 Для убытия в "Другое" - ручной ввод места

### 👑 **Для админа:**
1. 🔧 "Админ-панель" - вход в систему управления
2. 📊 "Быстрая сводка" - просмотр текущего состояния
3. 📢 "Массовое уведомление" - отправка сообщений
4. 📥 "Экспорт журнала" - выгрузка данных

## 🚀 **Установка и запуск**

### 📋 **Требования:**
```bash
python-telegram-bot==20.7
python-dotenv==1.0.0
pandas==2.1.4
openpyxl==3.1.2
pytz==2023.3
```

### ⚙️ **Настройка:**
1. 📝 Создать `.env` файл с настройками
2. 🔑 Указать токен бота и ID админов
3. 🕐 Настроить часовой пояс
4. 🔔 Включить уведомления

### 🚀 **Запуск:**
```bash
python main.py
```
