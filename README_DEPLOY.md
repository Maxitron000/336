# 🚀 Развертывание Telegram бота на PythonAnywhere

## ✅ Исправленные проблемы:
- Ошибка импорта `get_user` в `utils.py` - исправлена
- Совместимость с Python 3.10 - настроена
- Интервал перезапуска - оптимизирован до 55 минут

## 📋 Быстрый старт:

### 1. Настройка окружения
```bash
# Загрузите проект в /home/yourusername/mysite
# Замените yourusername на ваше имя пользователя
```

### 2. Конфигурация
Отредактируйте `.env`:
```
BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=ваш_telegram_id
```

### 3. Развертывание
```bash
python3 deploy_pythonanywhere.py
```

### 4. Запуск 24/7
```bash
python3 task_scheduler.py
```

## 🔧 Особенности:
- Перезапуск каждые 55 минут
- Python 3.10 совместимость
- Автоматическое создание папок
- Проверка БД перед запуском
