# 🚀 БЫСТРЫЙ ЗАПУСК TELEGRAM БОТА

## Что нужно сделать

### 1. Получить данные для бота
- Напишите [@BotFather](https://t.me/BotFather) и создайте нового бота
- Скопируйте токен (выглядит как `123456789:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`)
- Напишите [@userinfobot](https://t.me/userinfobot) и узнайте свой ID

### 2. Запустить установку
```bash
./quick_start.sh
```

### 3. Ввести данные
- Токен бота
- Ваш Telegram ID

### 4. Ждать 30 секунд
Всё настроится автоматически!

## Готово! 🎉

Бот работает 24/7 и готов к использованию.
Найдите его в Telegram и напишите `/start`

## Управление ботом

### Проверить статус
```bash
sudo systemctl status telegram-bot
```

### Перезапустить
```bash
sudo systemctl restart telegram-bot
```

### Остановить
```bash
sudo systemctl stop telegram-bot
```

### Посмотреть логи
```bash
tail -f logs/bot.log
```

## Если что-то пошло не так

### Проверить систему
```bash
python3 test_setup.py
```

### Переустановить
```bash
sudo systemctl stop telegram-bot
./quick_start.sh
```
