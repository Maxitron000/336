# 🤖 Как запустить бота - Инструкция для новичков

## 🎯 Что нужно сделать? Всего 4 шага!

### Шаг 1: 📝 Получить токен бота
1. Напишите в Telegram боту @BotFather
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например: "Мой учетный бот")
4. Придумайте username (например: "my_accounting_bot")
5. Скопируйте токен (длинная строка типа: `1234567890:ABCDEF...`)

### Шаг 2: 🆔 Узнать свой Telegram ID
1. Напишите боту @userinfobot
2. Отправьте любое сообщение
3. Скопируйте ваш ID (число типа: `123456789`)

### Шаг 3: ⚙️ Настроить файл .env
1. Откройте файл `.env` в блокноте
2. Замените `YOUR_BOT_TOKEN_HERE` на токен из шага 1
3. Замените `YOUR_TELEGRAM_ID_HERE` на ID из шага 2
4. Замените `yourusername` на ваше имя пользователя на PythonAnywhere

**Пример готового файла .env:**
```
BOT_TOKEN=1234567890:ABCDEF_ваш_токен_бота
ADMIN_IDS=123456789
PYTHONPATH=/home/ivan/mysite
```

### Шаг 4: 🚀 Запустить бота
1. Загрузите все файлы на PythonAnywhere в папку `/home/yourusername/mysite`
2. Откройте консоль на PythonAnywhere
3. Выполните команды по порядку:

```bash
# Установка и настройка (делается один раз)
python3 1_setup.py

# Запуск бота навсегда (работает 24/7)
python3 3_auto_restart.py
```

## 🎉 Готово! Бот работает!

### 📱 Что дальше?
- Напишите боту в Telegram
- Он попросит вас зарегистрироваться
- Введите ваше ФИО в формате: `Иванов И.И.`
- Теперь можете отмечать приходы и уходы!

### 🔄 Автоматический перезапуск
- Бот перезапускается каждые 55 минут
- Это нужно для стабильной работы на PythonAnywhere
- Вы ничего не заметите, бот продолжит работать

### 🆘 Если что-то не работает:
1. Проверьте правильность токена в `.env`
2. Проверьте правильность вашего ID в `.env`
3. Убедитесь, что заменили `yourusername` на ваше имя
4. Напишите боту @BotFather команду `/mybots` и проверьте, что бот включен

### 📋 Список файлов (что за что отвечает):
- `1_setup.py` - настройка и установка (запускать один раз)
- `2_start.sh` - запуск бота (можно запускать вручную)
- `3_auto_restart.py` - автоматический перезапуск (запускать для 24/7)
- `.env` - настройки бота (токен, ID админа)
- `main.py` - основная программа бота
- `requirements.txt` - список нужных программ
