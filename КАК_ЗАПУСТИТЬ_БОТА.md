# 🤖 КАК ЗАПУСТИТЬ TELEGRAM БОТА

## 🎯 БЫСТРЫЙ СТАРТ

### 1. Получите токен бота
1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и скопируйте токен (выглядит как `1234567890:ABCDEF...`)

### 2. Получите ваш Telegram ID
1. Найдите бота [@userinfobot](https://t.me/userinfobot)
2. Отправьте любое сообщение
3. Скопируйте ваш ID (число вида `123456789`)

### 3. Настройте конфигурацию
Отредактируйте файл `.env` в корне проекта:
```bash
# Замените YOUR_BOT_TOKEN на реальный токен
BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZ

# Замените YOUR_ADMIN_ID на ваш Telegram ID
ADMIN_IDS=123456789
```

### 4. Запустите бота
```bash
# Активируйте виртуальное окружение
source venv/bin/activate

# Запустите бота
python main.py
```

## ✅ ГОТОВО!

Если все настроено правильно, бот запустится и отправит сообщение в консоль:
```
🤖 Бот запущен успешно!
📱 Найдите вашего бота в Telegram и отправьте /start
```

## 📋 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### Ошибка "BOT_TOKEN не настроен"
- Проверьте, что токен правильно указан в файле `.env`
- Убедитесь, что нет лишних пробелов

### Ошибка "python-dotenv не установлен"
- Это нормально, модуль установлен под другим именем
- Игнорируйте это сообщение

### Бот не отвечает
- Убедитесь, что токен правильный
- Проверьте, что бот не заблокирован у @BotFather

## 🆘 ПОДДЕРЖКА

Если возникли проблемы:
1. Проверьте файл логов: `logs/bot.log`
2. Убедитесь, что все зависимости установлены
3. Проверьте конфигурацию командой `python check_config.py`
