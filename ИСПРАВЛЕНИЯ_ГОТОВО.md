1. **Получить токен бота у @BotFather в Telegram**
2. **Получить ваш Telegram ID у @userinfobot**
3. **Отредактировать файл .env:**
   ```bash

   BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZ
   
   # Замените YOUR_ADMIN_ID на ваш Telegram ID
   ADMIN_IDS=123456789
   ```
