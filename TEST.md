# 🧪 Тестирование функций бота

## Предварительная проверка

1. **Проверьте конфигурацию:**
   ```bash
   python3 tools.py check
   ```

2. **Создайте резервную копию:**
   ```bash
   python3 tools.py backup
   ```

3. **Запустите бота:**
   ```bash
   ./start_bot.sh
   ```

## Тестирование пользовательских функций

### 1. Регистрация нового пользователя
- Отправьте `/start` боту
- Введите ФИО в формате "Тестов Т.Т."
- ✅ Ожидается: Успешная регистрация и главное меню

### 2. Отметка прибытия
- Нажмите "🏠 Прибыл в расположение"
- ✅ Ожидается: Подтверждение и возврат в меню

### 3. Отметка убытия
- Нажмите "🚶‍♂️ Убыл"
- Выберите любую локацию (например, "🏥 Поликлиника")
- Выберите "✅ Добавить комментарий" или "❌ Без комментария"
- Если добавляете комментарий, введите текст
- ✅ Ожидается: Подтверждение убытия

### 4. Журнал действий
- Нажмите "📜 История действий"
- Нажмите "Показать 10"
- ✅ Ожидается: Список последних действий

### 5. Защита от двойных отметок
- Попробуйте нажать "🚶‍♂️ Убыл" снова, не отметившись прибывшим
- ✅ Ожидается: Сообщение об ошибке

## Тестирование админских функций

### 1. Вход в админку
- Отправьте `/admin` или нажмите "👑 Админ-панель"
- ✅ Ожидается: Админское меню

### 2. Управление личным составом
- Нажмите "👥 Управление ЛС"
- Нажмите "📋 Просмотр списка"
- ✅ Ожидается: Список всех пользователей

### 3. Просмотр журнала
- Нажмите "📖 Журнал и экспорт"
- Нажмите "📜 Журнал за сегодня"
- ✅ Ожидается: Записи за текущий день

### 4. Экспорт данных
- В разделе "📖 Журнал и экспорт"
- Нажмите "💾 Экспорт за сегодня"
- Выберите формат (CSV, Excel или PDF)
- ✅ Ожидается: Файл отправлен в чат

### 5. Сводка
- Нажмите "📊 Сводка"
- ✅ Ожидается: Статистика по локациям

## Тестирование уведомлений

### 1. Уведомления админу
- При любом действии пользователя
- ✅ Ожидается: Админ получает уведомление с деталями

### 2. Ежедневная сводка (19:00)
- Дождитесь 19:00 или измените время в notifications.py
- ✅ Ожидается: Автоматическая сводка админу

### 3. Вечерние напоминания (20:30)
- Дождитесь 20:30 или измените время
- ✅ Ожидается: Напоминание неотметившимся пользователям

## Тестирование утилит

### 1. Статистика
```bash
python3 tools.py stats
```
✅ Ожидается: Отчет по пользователям и действиям

### 2. Список пользователей
```bash
python3 tools.py users
```
✅ Ожидается: Таблица всех пользователей

### 3. Добавление админа
```bash
python3 tools.py add-admin --tg-id 123456789
```
✅ Ожидается: Подтверждение добавления

### 4. Очистка старых логов
```bash
python3 tools.py clean --days 7
```
✅ Ожидается: Отчет об удаленных записях

## Тестирование безопасности

### 1. Защита от быстрых нажатий
- Быстро нажмите одну кнопку несколько раз
- ✅ Ожидается: Обработка только первого нажатия

### 2. Валидация ФИО
- Попробуйте ввести неправильный формат: "Иванов Иван"
- ✅ Ожидается: Сообщение об ошибке формата

### 3. Права доступа
- Войдите как обычный пользователь
- Попробуйте отправить `/admin`
- ✅ Ожидается: Сообщение об отсутствии прав

### 1. Множественные пользователи
- Создайте несколько тестовых аккаунтов
- Отмечайтесь одновременно
- ✅ Ожидается: Все действия обрабатываются корректно

## Восстановление после сбоев

### 1. Перезапуск бота
- Остановите бота (Ctrl+C)
- Запустите снова
- ✅ Ожидается: Все данные сохранены, бот работает

### 2. Восстановление базы
- Создайте резервную копию
- Удалите базу данных
- Восстановите из копии
- ✅ Ожидается: Все данные восстановлены
