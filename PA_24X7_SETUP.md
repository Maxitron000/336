## 🚀 Быстрый старт

### 1. Первоначальная настройка
```bash
# Запустите базовую настройку
python3.10 setup.py

# Настройте для PythonAnywhere
python3.10 pythonanywhere_deploy.py
```

### 2. Настройка Scheduled Task в PythonAnywhere

1. Откройте **PythonAnywhere Dashboard**
2. Перейдите в раздел **"Tasks" → "Scheduled"**
3. Нажмите **"Create a scheduled task"**
4. Заполните поля:

```
Command: /home/yourusername/telegram_bot/venv_bot/bin/python /home/yourusername/telegram_bot/pa_runner.py
Hour: * (звездочка - каждый час)
Minute: 0
Description: Telegram Bot 24/7 Monitor
```

**❗ Важно:** Замените `yourusername` на ваше реальное имя пользователя!

### 3. Альтернативные способы запуска

#### Вариант A - Прямой запуск монитора (рекомендуется)
```bash
python3.10 pa_24x7_monitor.py
```

#### Вариант B - Через обертку
```bash
python3.10 pa_runner.py
```

#### Вариант C - Bash скрипт
```bash
./start_pa.sh
```

## 🔧 Решение проблем

### Бот не запускается
1. Проверьте `.env` файл: `cat .env`
2. Проверьте виртуальное окружение: `ls -la venv_bot/`
3. Проверьте зависимости: `venv_bot/bin/pip list`
4. Запустите диагностику: `python3.10 diagnose_issues.py`

### Нет уведомлений
1. Проверьте токен бота в `.env`
2. Проверьте ID админа в `.env`
3. Убедитесь что бот имеет права писать пользователю

### Высокое потребление ресурсов
1. Проверьте логи: `tail -f logs/pa_monitor.log`
2. Уменьшите частоту проверок в конфигурации
3. Оптимизируйте запросы к базе данных
