# 🚀 Быстрый старт: Военная версия бота

## 📋 Краткое описание

Телеграм-бот для учета личного состава военной части с:
- 11 локаций (включая "Другое" с вводом текста)
- Система уведомлений (2 раза в день)
- 50 шуточных фраз в военной тематике
- Административная панель с настройками

## ⚙️ Настройка

### 1. Установка зависимостей
```bash
pip install -r requirements_optimized.txt
```

### 2. Настройка .env файла
```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_IDS=123456789,987654321
```

### 3. Запуск
```bash
python3 main_optimized.py
```

## 🔔 Система уведомлений

### 📊 Сводка админам (19:00):
- Отчет о местонахождении всех
- 🟢 В расположении (ОБРМП, Рабочка, Столовая)
- 🔴 Не в расположении (с локациями)
- ⚫ Без отметок

### 💬 Напоминания (20:30):
- Только тем, кто убыл но не вернулся
- Случайная фраза из 50 военных шуток
- Примеры:
  - "Бро, где ты? 🤔 Отметься, а то думаем ты в плену! 😄"
  - "Товарищ, связь потеряна! 📡 Дай сигнал о своем местонахождении! 🚨"

## ⚙️ Настройки

### Админская панель:
- Включение/отключение сводки
- Включение/отключение напоминаний
- Настройка времени (по умолчанию 19:00 и 20:30)

### Код настроек:
```python
NOTIFICATION_SETTINGS = {
    'summary_time': '19:00',
    'reminder_time': '20:30',
    'summary_enabled': True,
    'reminder_enabled': True
}
```
