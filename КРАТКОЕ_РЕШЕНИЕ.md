# 🚨 Быстрое решение ошибки подключения к Telegram API

## Проблема
```
ClientConnectorError: Cannot connect to host api.telegram.org:443 ssl:default
```

## Причина
❌ Не настроен файл `.env` с токеном бота

## Решение (2 минуты)

### 1. Создайте бота в Telegram
- Откройте [@BotFather](https://t.me/BotFather)
- Отправьте `/newbot`
- Скопируйте токен

### 2. Получите свой ID
- Откройте [@userinfobot](https://t.me/userinfobot)
- Скопируйте ваш ID

### 3. Настройте .env
Замените в файле `.env`:
```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id
```

### 4. Проверьте настройки
```bash
python3 check_config.py
```

### 5. Запустите бота
```bash
python3 main.py
```

## ✅ Готово!
Бот должен работать без ошибок.
