# 🚀 Быстрый запуск бота в локальной среде

## ⚡ Всего 3 шага до запуска!

### Шаг 1: 🔧 Настройте токен бота
1. Напишите в Telegram боту @BotFather
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например: "Мой учетный бот")
4. Придумайте username (например: "my_accounting_bot")
5. Скопируйте токен (длинная строка типа: `1234567890:ABCDEF...`)

### Шаг 2: 🆔 Получите свой Telegram ID
1. Напишите боту @userinfobot
2. Отправьте любое сообщение
3. Скопируйте ваш ID (число типа: `123456789`)

### Шаг 3: ⚙️ Настройте файл .env
Откройте файл `.env` и замените:
- `YOUR_BOT_TOKEN_HERE` на токен из шага 1
- `YOUR_TELEGRAM_ID_HERE` на ID из шага 2

**Пример:**
```env
BOT_TOKEN=1234567890:ABCDEF_ваш_токен_бота
ADMIN_IDS=123456789
```

## 🏃 Запуск бота

### Автоматическая настройка:
```bash
python3 setup_local.py
```

### Запуск бота:
```bash
./start_bot.sh
```

## 🎉 Готово!

Ваш бот теперь работает! Найдите его в Telegram по username и напишите `/start`.

## 🔧 Если возникли проблемы:

### Проблема: "Permission denied"
```bash
chmod +x start_bot.sh
./start_bot.sh
```

### Проблема: "BOT_TOKEN не найден"
- Проверьте правильность токена в файле `.env`
- Убедитесь, что удалили `YOUR_BOT_TOKEN_HERE`

### Проблема: "python3 не найден"
```bash
# Для Ubuntu/Debian:
sudo apt install python3 python3-pip python3-venv

# Для CentOS/RHEL:
sudo yum install python3 python3-pip
```

### Проблема: "Бот не отвечает"
1. Проверьте, что бот запущен (должно быть сообщение "✅ Бот успешно запущен!")
2. Убедитесь, что ваш ID указан в ADMIN_IDS
3. Попробуйте команду `/start` в Telegram

## 📋 Полезные команды:

- `/start` - Главное меню
- `/help` - Справка
- `/status` - Статус системы
- `/admin` - Админ-панель (только для админов)

## 🛑 Остановка бота:

Нажмите `Ctrl+C` в терминале где запущен бот.

## 📊 Мониторинг:

Логи сохраняются в папке `logs/bot.log`
```bash
tail -f logs/bot.log
```
