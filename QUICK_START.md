# 🚀 Быстрый старт Telegram бота

## Что нужно для запуска:
1. **Токен бота** - получите у @BotFather
2. **Ваш Telegram ID** - узнайте у @userinfobot

## Установка и запуск (одной командой):

```bash
python setup.py
```

Скрипт автоматически:
- ✅ Создаст виртуальное окружение `venv_bot/`
- ✅ Установит все зависимости
- ✅ Попросит ввести токен бота и ваш ID
- ✅ Настроит конфигурацию
- ✅ Инициализирует базу данных

## Запуск бота:

```bash
python start.py
```

## Требования:
- Python 3.10+
- Интернет для установки зависимостей

## Если что-то пошло не так:
1. Убедитесь что Python 3.10+
2. Проверьте интернет соединение
3. Запустите `python setup.py` заново
4. Все настройки сохранятся, нужно будет только переустановить зависимости

## После установки:
- Бот автоматически перезапускается каждый час
- Отчёты отправляются админу каждые 6 часов
- Все логи в файле `logs/bot.log`
- Конфигурация в файле `.env`

**Готово! Больше ничего настраивать не нужно! 🎉**
