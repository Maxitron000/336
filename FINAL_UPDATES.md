## 🔔 Система уведомлений

### Типы уведомлений:
1. **Утренние** (настраиваемое время) - напоминание отметиться
2. **Вечерние** (настраиваемое время) - для не отметившихся
3. **Периодические** (настраиваемый интервал) - "братские" напоминания
4. **Административные** - для админов о действиях пользователей
5. **Системные** - важные уведомления всем

### Настройка через переменные окружения:
```env
NOTIFICATION_MORNING_TIME=08:00
NOTIFICATION_EVENING_TIME=18:00
NOTIFICATION_INTERVAL_HOURS=4
ENABLE_NOTIFICATIONS=true
```

## 🗄️ База данных

### Структура таблиц:
- `users` - Пользователи (id, telegram_id, full_name, username, is_admin, registered_at, last_activity)
- `location_logs` - Журнал местоположений (id, user_id, location, action, timestamp)
- `active_sessions` - Активные сессии (id, user_id, location, entered_at)
- `admin_logs` - Административные логи (id, admin_id, action, target_user_id, details, timestamp)

### Функции базы данных:
- Автоматическое создание и инициализация
- Проверка целостности данных
- Оптимизация производительности
- Резервное копирование и восстановление
