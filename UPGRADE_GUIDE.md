## 📦 Установка обновления

### 1. Резервное копирование
```bash
# Создайте резервную копию текущей базы данных
cp data/bot_database.db data/bot_database_backup_$(date +%Y%m%d_%H%M%S).db
```

### 2. Обновление файлов
Замените следующие файлы на новые версии:
- `database.py` - обновленная структура БД
- `notifications.py` - новая система уведомлений
- `keyboards.py` - обновленные клавиатуры
- `handlers.py` - новые обработчики
- `migrate_database.py` - скрипт миграции (новый)

### 3. Создание новых файлов
```bash
# Создайте папку config если её нет
mkdir -p config

# Скопируйте файл с шаблонами уведомлений
cp config/notifications.json config/notifications.json
```

### 4. Миграция базы данных
```bash
# Запустите скрипт миграции
python3 migrate_database.py
```

### 5. Перезапуск бота
```bash
# Остановите текущий бот
pkill -f "python3 main.py"

# Запустите обновленную версию
python3 main.py
```

### Валидация ФИО

## 🆘 Решение проблем

### Ошибка миграции
```bash
# Восстановите из резервной копии
cp data/bot_database_backup_YYYYMMDD_HHMMSS.db data/bot_database.db
```

### Проблемы с уведомлениями
1. Проверьте файл `config/notifications.json`
2. Убедитесь, что командиры имеют PUSH-права
3. Проверьте настройки уведомлений в админ-панели

### Проблемы с валидацией
1. Проверьте формат ФИО пользователей
2. Убедитесь, что нет дублирующихся записей
3. Проверьте логи на наличие ошибок
