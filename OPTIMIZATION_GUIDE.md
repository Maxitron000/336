## 🔄 Алгоритм автоочистки

### 1. Очистка по расписанию:
- Каждый день в 3:00 UTC
- Удаление записей старше 6 месяцев
- Оптимизация БД (VACUUM)

### 2. Очистка при превышении лимитов:
- Если БД > 50MB - принудительная очистка
- Если логи > 5MB - обрезка файла
- Если общий размер > 400MB - экстренная очистка

### 3. Мониторинг размера:
```python
# Проверка каждые 24 часа
def check_disk_usage():
    total_size = get_directory_size('.')
    if total_size > 400 * 1024 * 1024:  # 400MB
        emergency_cleanup()
```
