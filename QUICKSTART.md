# ⚡ Быстрый старт за 5 минут

Этот гайд поможет вам запустить бота для учета персонала за 5 минут.

## 🚀 Шаг 1: Создание бота (1 минута)

### В Telegram:
1. Найдите @BotFather
2. Отправьте `/newbot`
3. Выберите имя: `Personnel Bot`
4. Выберите username: `your_personnel_bot`
5. **Сохраните токен** вида `1234567890:AABBCCDDEEFFgghhiijjkkllmmnnoopp`

### Получите свой Telegram ID:
1. Найдите @userinfobot
2. Отправьте любое сообщение
3. **Сохраните ID** вида `123456789`

## 💻 Шаг 2: Установка (2 минуты)

```bash
# Клонируем проект
git clone https://github.com/your-username/telegram-personnel-bot.git
cd telegram-personnel-bot

# Устанавливаем зависимости
pip install -r requirements.txt

# Создаем конфигурацию
cp .env.example .env
```

## ⚙️ Шаг 3: Настройка (1 минута)

Отредактируйте файл `.env`:

```env
# Вставьте ваш токен от BotFather
BOT_TOKEN=1234567890:AABBCCDDEEFFgghhiijjkkllmmnnoopp

# Вставьте ваш Telegram ID
ADMIN_IDS=123456789

# Остальные настройки можно оставить по умолчанию
TIMEZONE=Europe/Kaliningrad
ENABLE_NOTIFICATIONS=true
```

## 🏃 Шаг 4: Запуск (1 минута)

```bash
# Делаем скрипт исполняемым
chmod +x start_bot.sh

# Запускаем бота
./start_bot.sh
```

Если видите сообщение "🚀 Бот запущен и готов к работе!" - всё готово!

## ✅ Шаг 5: Проверка (30 секунд)

1. Найдите своего бота в Telegram
2. Отправьте `/start`
3. Зарегистрируйтесь: `Петров П.П.`
4. Попробуйте отметить местоположение
5. Войдите в админ-панель через `🔧 Админ-панель`

## 🚨 Решение проблем

### Ошибка: "Invalid token"
- Проверьте токен в `.env` файле
- Убедитесь, что копировали полностью

### Ошибка: "No module named 'telegram'"
```bash
pip install python-telegram-bot==20.7
```

### Бот не отвечает
- Проверьте интернет-соединение
- Убедитесь, что токен корректный
- Попробуйте перезапустить: `./start_bot.sh`

### Не работают уведомления
- Проверьте `ENABLE_NOTIFICATIONS=true` в `.env`
- Убедитесь, что время уведомлений корректное
