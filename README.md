# 🤖 Personnel Management Bot

Интеллектуальный Telegram бот для управления персоналом с расширенными возможностями администрирования и автоматизации.

## ✨ Основные возможности

### 👥 Управление персоналом
- Добавление, редактирование и удаление сотрудников
- Поиск по различным критериям (имя, должность, отдел)
- Экспорт данных в Excel
- Статистика и аналитика

### 🔧 Админ-панель
- Многоуровневое меню с эмодзи
- Управление персоналом
- Журнал событий
- Настройки системы
- Зона опасности с подтверждениями

### 🔔 Система уведомлений
- Автоматические напоминания
- Ежедневные сводки
- Проверки здоровья системы
- Отчеты о производительности
- Гибкие настройки времени

### 📊 Аналитика
- Статистика по отделам
- Анализ производительности
- Отчеты в реальном времени
- Визуализация данных

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd personnel-bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка конфигурации
Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

Заполните необходимые параметры:
```env
BOT_TOKEN=your_bot_token_here
ADMIN_ID=your_telegram_id
ADMIN_IDS=id1,id2,id3
```

### 4. Запуск бота
```bash
python main.py
```

## 🔧 Конфигурация

### Переменные окружения (.env)
- `BOT_TOKEN` - токен Telegram бота
- `ADMIN_ID` - ID главного администратора
- `ADMIN_IDS` - список ID администраторов
- `DAILY_SUMMARY_TIME` - время ежедневной сводки
- `REMINDERS_TIME` - время напоминаний
- `HEALTH_CHECK_TIME` - время проверки здоровья
- `PERFORMANCE_REPORT_TIME` - время отчета о производительности

### Настройки уведомлений (notifications.json)
```json
{
  "daily_summary": {
    "enabled": true,
    "time": "19:00",
    "message": "� Ежедневная сводка"
  },
  "reminders": {
    "enabled": true,
    "time": "20:30",
    "message": "⏰ Напоминание"
  }
}
```

## 🎯 Использование

### Для пользователей
1. Отправьте `/start` для начала работы
2. Используйте меню для навигации
3. Следуйте инструкциям на экране

### Для администраторов
1. Отправьте `/admin` для доступа к админ-панели
2. Используйте многоуровневое меню
3. Настройте автоматические уведомления
4. Просматривайте статистику и отчеты

## 🔒 Безопасность

- Все конфиденциальные данные хранятся в `.env`
- Доступ к админ-панели только для авторизованных пользователей
- Подтверждения для опасных операций
- Логирование всех действий
