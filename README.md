# 🤖 Telegram Bot для учета персонала

Telegram бот для отслеживания прибытия/убытия персонала с административными функциями и подробной отчетностью.

## ✅ Исправленные проблемы

- ✅ Конфликты версий aiogram 2.25.1 с Python 3.13
- ✅ Проблемы с базой данных  
- ✅ Автоматическая диагностика и исправление зависимостей
- ✅ Оптимизация для PythonAnywhere

## � Быстрый старт

### 1. Настройка конфигурации

```bash
# Скопируйте и настройте файл конфигурации
cp .env.example .env
nano .env

# Замените в .env файле:
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id
```

### 2. Установка зависимостей

**Для Python 3.10 (рекомендуется):**
```bash
# Создайте виртуальное окружение
python3.10 -m venv venv_bot
source venv_bot/bin/activate

# Установите зависимости
pip install -r requirements.txt
```

**Автоматическое исправление проблем:**
```bash
python fix_dependencies.py
```

### 3. Диагностика

```bash
# Проверьте состояние всех компонентов
python diagnose_issues.py
```

### 4. Запуск бота

```bash
# Локальный запуск
python main.py

# Для PythonAnywhere (оптимизированный)
python run_bot.py
```

## 🔧 Настройка на PythonAnywhere

### Пошаговая инструкция

1. **Загрузите проект:**
   ```bash
   git clone YOUR_REPOSITORY_URL telegram_bot
   cd telegram_bot
   ```

2. **Используйте Python 3.10:**
   ```bash
   python3.10 -m venv venv_bot
   source venv_bot/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте конфигурацию:**
   ```bash
   cp .env.example .env
   nano .env
   # Добавьте реальные токены
   ```

5. **Запустите диагностику:**
   ```bash
   python diagnose_issues.py
   ```

6. **Настройте Scheduled Task:**
   - Command: `/home/yourusername/telegram_bot/venv_bot/bin/python /home/yourusername/telegram_bot/run_bot.py`
   - Hour: `*` (каждый час)
   - Minute: `0`

### Особенности для бесплатного аккаунта

- **Рабочие часы:** 6:00-23:00 (экономия CPU времени)
- **Автоперезапуск:** Каждый час через Scheduled Task
- **Оптимизация памяти:** Автоматический контроль использования ресурсов

## � Требования

### Системные требования
- **Python:** 3.8-3.10 (рекомендуется 3.10)
- **ОС:** Linux, macOS, Windows
- **Память:** минимум 512MB
- **Диск:** 100MB свободного места

### Python пакеты
- `aiogram==2.25.1` - Telegram Bot API
- `aiosqlite==0.19.0` - Асинхронная работа с SQLite
- `python-dotenv==1.0.0` - Переменные окружения
- `aiohttp>=3.8.0,<3.9.0` - HTTP клиент
- `pandas`, `openpyxl` - Работа с Excel
- `reportlab` - Генерация PDF

Полный список в `requirements.txt`

## 📞 Поддержка

### При проблемах:

1. **Запустите диагностику:**
   ```bash
   python diagnose_issues.py
   ```

2. **Проверьте логи:**
   ```bash
   tail -50 logs/bot.log
   ```

3. **Автоматическое исправление:**
   ```bash
   python fix_dependencies.py
   ```

4. **Проверьте конфигурацию:**
   ```bash
   cat .env
   ```

### Частые проблемы

**ModuleNotFoundError: aiogram**
```bash
# Решение
python fix_dependencies.py
```

**База данных не создается**
```bash
# Решение  
python create_database.py
```

**Бот не отвечает**
```bash
# Проверьте токен
grep BOT_TOKEN .env
```
