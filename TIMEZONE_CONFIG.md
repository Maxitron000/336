# ⏰ Настройка часового пояса

## 🌍 Текущая настройка

По умолчанию система использует **часовой пояс Калининграда** (UTC+2), как указано в ТЗ.

## 🔧 Как изменить часовой пояс

### 1. Отредактируйте файл `.env`
```bash
nano .env
```

### 2. Измените переменную TIMEZONE
```env
# Примеры популярных часовых поясов России:

# Калининград (UTC+2) - по умолчанию
TIMEZONE=Europe/Kaliningrad

# Москва (UTC+3)
TIMEZONE=Europe/Moscow

# Екатеринбург (UTC+5)
TIMEZONE=Asia/Yekaterinburg

# Новосибирск (UTC+7)
TIMEZONE=Asia/Novosibirsk

# Иркутск (UTC+8)
TIMEZONE=Asia/Irkutsk

# Владивосток (UTC+10)
TIMEZONE=Asia/Vladivostok
```

### 3. Перезапустите бота
```bash
./start_bot.sh
```

| Регион | Часовой пояс | UTC |

| 🏛️ Москва | `Europe/Moscow` | UTC+3 |

## ✅ Проверка настройки

После изменения часового пояса проверьте:

1. **Время в журналах** - должно отображаться корректно
2. **Автоматические уведомления** - приходят в нужное время
3. **Экспорт данных** - временные метки корректны

## 🛠️ Если что-то пошло не так

### Ошибка "Unknown timezone"
```bash
# Проверьте правильность написания
python3 -c "import pytz; print(pytz.timezone('YOUR_TIMEZONE'))"
```

### Вернуть настройки по умолчанию
```env
TIMEZONE=Europe/Kaliningrad
```

### Посмотреть текущее время системы
```bash
python3 -c "from utils import get_current_time; print(get_current_time())"
```

## 📋 Заметки

- Изменение часового пояса не влияет на уже сохраненные данные
- Все новые записи будут использовать новый часовой пояс
- Автоматические уведомления (19:00, 20:30) будут приходить по новому времени
- Экспорт данных будет показывать время в новом часовом поясе

**⚠️ Важно:** После изменения часового пояса обязательно перезапустите бота!
